# coxu

Yet another calculator that supports *base 3*

## Usage
1. `stack run` (or run binary)
2. Enter the expression (supports multiline expressions, c-style comments)
3. Press `Ctrl + D` on Linux, `Ctrl + Z` on Windows

```
$ stack run
(1 + /* comment */
  (2 + 3))
  Base 10: 6
  Base 3: +-0
```
